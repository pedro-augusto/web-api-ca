## Overview

This is a full-stack movie web application built using React, Node.js, Express, and MongoDB.  
The application allows users to browse movies and view detailed information retrieved through a custom backend API that communicates with the TMDB service.

Users can create an account, log in, and manage personal movie lists. All user-specific data is securely stored and only accessible when logged in.

---

## Features

- Browse movies using discover, trending, top-rated, upcoming, and now-playing views  
- Search for movies by keyword  
- View detailed movie information including credits, reviews, recommendations, and images  
- User registration and login with password validation  
- Protected pages accessible only to logged-in users  
- Add and remove movies from Favourites and Must Watch lists  
- User-specific data (each user sees only their own interactions)  
- Dynamic interface changes based on authentication status
